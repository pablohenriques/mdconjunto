# Matemática Discreta - Conjunto de Dados

 ## Aplicação (Contexto)
 O projeto desenvolvido é uma API (Aplication Programming Interface / Interface de Programação de Aplicações) para criação de conjuntos. Este projeto foi produzido como parte da obtenção de nota na disciplina de Matemática Discreta, no CEULP/ULBRA, ministrada pelo professora Jackson Gomes.
